Tutorial

Select an event and a Bible translation, then press Submit. You can adjust text size and line spacing using the sliders. To rearrange the Gospel boxes, click and drag by the author's name. Each box can also be resized vertically to fit more text. Found a bug? Let me know on GitHub !
